## 🚀 Featured Project

### 🔩 [BareMetalML](https://github.com/bhaskar9221/BareMetalML) &nbsp;![Status](https://img.shields.io/badge/status-active-brightgreen?style=flat-square)

> *An open-source ML library built from the ground up using only NumPy — no scikit-learn, no shortcuts.*

Implementing every algorithm end-to-end with full mathematical transparency:

- ✅ **Linear Regression** — Normal Equation, Batch/SGD, Ridge (L2) regularization
- ✅ **Logistic Regression** — Sigmoid, Binary Cross-Entropy, One-vs-Rest multiclass
- ✅ **Decision Trees** — Gini Impurity / Entropy splitting, decision boundary visualization
- 🔄 **Random Forest** — Bootstrap Aggregation (Bagging) *(in progress)*
- 🏗️ Modular architecture: custom train/test split, StandardScaler, MinMaxScaler, MSE/R² metrics

`Python` · `NumPy` · `Pandas` · `Matplotlib`
